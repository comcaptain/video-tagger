暂时只支持PotPlayer

### 1. 设置截图

鼠标右键 -> 视频 -> 图像截取 ->
- 选择`以播放时间命名文件`
- 设置截图路径为`F:\video-tagger-data\potplayer-screenshot`

鼠标右键 -> 选项 -> 基本 -> 快捷键 -> 为`视频 -> 图像截取 -> 截存当前原画面`设置快捷键`Ctrl+Alt+Shift+E`

### 2. 设置复制文件完整路径快捷键

鼠标右键 -> 选项 -> 基本 -> 快捷键 -> 为`配置/语言/其他 -> 辅助处理 -> 将文件完整路径复制到剪贴板`设置快捷键`Ctrl+Alt+Shift+C`
